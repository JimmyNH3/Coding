1. 使用 `資料型態[] 陣列名稱` 宣告陣列：

   ```java
   int[] arr;
   ```
2. 使用 `陣列名稱 = new 資料型態[陣列大小]`初始化陣列

   ```java
   arr = new int[3];
   ```

   *以上兩者可合併：

   ```java
   int[] arr = new int[3];
   ```
3. 可在宣告陣列時同時存入資料：(`[陣列大小]`須為空)

   ```java
   int[] arr = new int[]{1, 2, 3};
   ```
4. 用 `arr.length`取得陣列長度
